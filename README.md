# Iris Flower Classification using Supervised Learning
## Project Overview

This project demonstrates a supervised machine learning approach to classify iris flowers into three species:

Setosa
Versicolor
Virginica

Using the famous Iris dataset, the model predicts the species based on flower measurements like sepal and petal dimensions.

## Dataset Information

The dataset contains the following features:

Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Target Variable)
## Objective

To build a machine learning model that:

Learns from labeled data
Accurately classifies iris flower species
Evaluates performance using standard metrics
